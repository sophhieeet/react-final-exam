# Travel Journal

React final project created by **Sopiko Baindurashvili**.

## idea

The application is a travel/country catalogue. Users can browse countries from a real API, open a modal with details, save countries to a personal list, switch theme and change language.

## Technologies

- React
- Vite
- Functional Components
- React Hooks: `useState`, `useEffect`, `useMemo`, `useContext`
- React Router DOM
- Axios
- LocalStorage
- SessionStorage
- SCSS
- Responsive Design
- Modal Window
- Dark / Light Theme
- Georgian / English Language

## Pages

1. Home
2. Destinations
3. Saved Trips
4. About Project
5. 404 page

## API

Country data is loaded from REST Countries API:

```txt
https://restcountries.com/v3.1/all
```

## Storage usage

- Theme is saved in LocalStorage
- Language is saved in LocalStorage
- Saved countries are saved in LocalStorage
- Last opened country is saved in SessionStorage

## How to run

```bash
npm install
npm run dev
```
